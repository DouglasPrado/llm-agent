# Referencia de API

Referencia completa de todos os tipos, interfaces e metodos exportados pelo AgentX SDK.

## Classe Agent

A classe principal do SDK. Ponto de entrada para todas as interacoes.

### Agent.create(config)

Cria e valida uma instancia do agente.

```typescript
static create(input: AgentConfigInput): Agent
```

| Parametro | Tipo | Descricao |
|-----------|------|-----------|
| `input` | `AgentConfigInput` | Configuracao do agente (validada com Zod) |

**Retorna:** `Agent`

**Lanca:** `ZodError` se a configuracao for invalida.

---

### agent.stream(input, options?)

API primaria de streaming. Retorna um iterador assincrono de eventos.

```typescript
async *stream(
  input: string | ContentPart[],
  options?: ChatOptions
): AsyncIterableIterator<AgentEvent>
```

| Parametro | Tipo | Descricao |
|-----------|------|-----------|
| `input` | `string \| ContentPart[]` | Mensagem do usuario (texto ou multimodal) |
| `options` | `ChatOptions` | Opcoes de execucao (opcional) |

**Retorna:** `AsyncIterableIterator<AgentEvent>`

---

### agent.chat(input, options?)

API simplificada. Consome `stream()` e retorna o texto final.

```typescript
async chat(
  input: string | ContentPart[],
  options?: ChatOptions
): Promise<string>
```

**Retorna:** `Promise<string>` — texto completo da resposta.

**Lanca:** `Error` se houver um erro irrecuperavel.

### agent.fork(prompt, options?)

Cria um agente filho que herda a config do pai, executa um chat isolado e retorna o resultado.

```typescript
async fork(
  prompt: string,
  options?: {
    systemPrompt?: string;
    model?: string;
    tools?: AgentTool[];
    background?: boolean;
  }
): Promise<string>
```

Se `background: true`, retorna `''` imediatamente (fire-and-forget).

## ChatOptions

Opcoes passadas para `stream()` e `chat()`:

```typescript
interface ChatOptions {
  threadId?: string;        // ID do thread (default: 'default')
  model?: string;           // Override do modelo para esta execucao
  temperature?: number;     // Temperatura para geracao
  signal?: AbortSignal;     // Signal para cancelamento
}
```

## AgentTool

Interface para definir uma ferramenta:

```typescript
interface AgentTool {
  name: string;
  description: string;
  parameters: ZodSchema;
  execute: (
    args: unknown,
    signal: AbortSignal,
    onProgress?: ToolProgressCallback,
  ) => Promise<string | AgentToolResult>;

  // Opcoes avancadas
  validate?: (args: unknown, context: ToolValidationContext) => Promise<string | null>;
  isConcurrencySafe?: boolean | ((args: unknown) => boolean);
  isReadOnly?: boolean | ((args: unknown) => boolean);
  isDestructive?: boolean | ((args: unknown) => boolean);
  getFilePath?: (args: unknown) => string | string[] | undefined;
  timeoutMs?: number;
  maxResultChars?: number;
  mapResult?: (result: AgentToolResult) => AgentToolResult;
  retryable?: boolean | ((error: unknown) => boolean);
  maxRetries?: number;
}
```

## AgentToolResult

Resultado retornado por uma tool:

```typescript
interface AgentToolResult {
  content: string;
  metadata?: Record<string, unknown>;
  isError?: boolean;
}
```

## AgentSkill

Interface para definir uma skill:

```typescript
interface AgentSkill {
  // Core
  name: string;
  description: string;
  instructions: string;

  // Matching
  match?: (input: string, context: SkillMatchContext) => boolean;
  triggerPrefix?: string;
  aliases?: string[];
  priority?: number;
  exclusive?: boolean;

  // Prompt
  getPrompt?: (args: string, context: SkillPromptContext) => Promise<string> | string;
  argNames?: string[];
  whenToUse?: string;

  // Execution
  tools?: AgentTool[];
  allowedTools?: string[];
  model?: string;
  context?: 'inline';
  effort?: number;

  // Session
  sticky?: boolean | number;

  // Activation
  paths?: string[];
  isEnabled?: () => boolean;
  modelInvocable?: boolean;

  // Metadata
  source?: 'programmatic' | 'directory' | 'mcp' | 'bundled';
  skillDir?: string;
}
```

## AgentEvent

Union type de todos os eventos de streaming:

```typescript
type AgentEvent =
  | AgentStartEvent
  | TextDeltaEvent
  | TextDoneEvent
  | ToolCallStartEvent
  | ToolCallDeltaEvent
  | ToolCallEndEvent
  | ToolProgressEvent
  | MemoryExtractedEvent
  | KnowledgeRetrievedEvent
  | SkillActivatedEvent
  | TurnStartEvent
  | TurnEndEvent
  | ErrorEvent
  | WarningEvent
  | CompactionEvent
  | RecoveryEvent
  | ModelFallbackEvent
  | AgentEndEvent;
```

Consulte a pagina de [Eventos](/pt/events) para a shape completa de cada evento.
